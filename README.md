# 欢迎使用 Photoflow
- 使用Electron+React+Python写的一个摄影人的工具包
- 代码基本上来自Google Gemini和Copilot
- 名字都是Gemini取的，写这个纯粹是应对一些日常需求提高效率而已
**软件没有经过充分测试。使用前请备份重要数据。作者不对因使用本软件造成的任何损失负责。**

## 功能特点

- **从SD卡导入图片**: 可以从SD卡自动导入图片，设置导出的地址，会直接剪切过来，然后以RAW底片，jpg图片和视频创建分类
- **设置喜欢的角色的生日在首页**: 一个很二次元的功能
- **PNG 转 JPG**: 偶尔有这个需求我干脆写里面了
- **选片**: 可以快速从文件名选取RAW文件出来，免得一张张找
- **整理前后期图片**: 可以对比两个文件夹里面的图片，可以匹配到稍微修过的图片
- **调研整理**: 从新媒体平台获取视频做调研的时候，这个工具可以把视频的转场中间帧提取出来，有GPU功能，但是GPU功能需要自编译的cv2，这个过程十分麻烦，没有GPU支持的时候会使用CPU渲染
